HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • HTML provides structure to a webpage.
  • HTML enables and provides standards for many types of web browsers.
  • With HTML you can create static or dynamic web pages.

CSS

  • CSS allows you to make individual or broad changes to colors and style.
  • We use CSS, which is a programming language used by developers, to define how webpage content is presented to users.
  • CSS defines the colors, aesthetics, visual layout, and organization.
  • We use CSS to define style rules for specific elements of an HTML page and format them using a wide range of style options.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
